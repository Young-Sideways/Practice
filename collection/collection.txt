Структуры данных:

Статические:
1.  variant - Структура, содержащая данные любго другого типа
2.  tuple   - Объединение нескольких полей разного типа, с возможностью итерации по ним
3.  array   - Массив
4.  matrix  - Матрица

Динамические:
1.  vector             - Динамический массив
    list
2.    +--> list        - Двусвязанный список
3.    +--> sorted_list - Отсортированный двусвязанный список
4    deque             - Двусторонняя очередь
5.    +--> queue       - Очередь
6.    +--> stack       - Стек
7.  set                - Набор, содержащий уникальные данные, основанный на хештаблице
8.  map                - Набор и пар "ключ-значение", основанный на хештаблице
    tree
9.    +--> AVL tree    - самобалансирующееся дерево
10.   +--> B tree      - самобалансирующееся дерево
11.   +--> RB tree     - самобалансирующееся дерево
